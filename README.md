## Installation 
### Dependencies 
- Python 3.6 
- Dash 0.21.0 

### How to Install 
```
***git clone repository*** not pip installable yet 
conda create -n py36 python=3.6 anaconda
source activate py36 
conda install -r requirements.txt 
```
